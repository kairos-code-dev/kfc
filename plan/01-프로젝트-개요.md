# 프로젝트 개요

## 프로젝트 정보

- **프로젝트명**: kfc (Korea Finance Collector)
- **패키지명**: `dev.kairoscode.kfc`
- **목적**: 한국 금융 데이터 수집 전용 Kotlin 라이브러리
- **라이센스**: MIT License

## 프로젝트 목표

kfc는 한국 금융시장 데이터를 수집하기 위한 **단일 목적 라이브러리**입니다. UNIX 철학의 "Do One Thing Well"을 따라 데이터 수집 기능에만 집중하며, 분석이나 시각화 기능은 제공하지 않습니다.

### 핵심 원칙

1. **데이터 수집 전문화**: 금융 데이터 수집 기능만 제공
2. **원본 데이터 반환**: API 응답을 가공 없이 그대로 반환
3. **1:1 API 매핑**: 각 함수는 하나의 API 엔드포인트에 대응
4. **소스 기반 분류**: 데이터 제공처별 패키지 구조 (KRX, Naver, OPENDART)

## 데이터 소스

kotlin-krx는 다음 3개의 공식 데이터 소스를 지원합니다:

| 데이터 소스 | 주요 제공 데이터 | 함수 개수 |
|------------|----------------|----------|
| **KRX** (한국거래소) | ETF 종목정보, 가격, 포트폴리오, 성과, 투자자별매매, 공매도, 수정주가 | 15개 |
| **Naver Finance** | ETF 보수율 정보 | 1개 |
| **OPENDART** (전자공시) | 기업정보, 재무제표, 배당정보, 주식발행 | 6개 |

**총 22개 API 함수 제공**

## 기술 스택

- **언어**: Kotlin 2.2.21
- **JDK**: Amazon Corretto 21
- **HTTP 클라이언트**: Ktor Client 3.3.2
- **직렬화**: Kotlinx Serialization 1.9.0
- **빌드 도구**: Gradle 9.2.1 (Kotlin DSL)
- **테스트**: Kotlin Test 2.2.21

## 아키텍처 개요

kfc는 **3계층 아키텍처**로 설계되었습니다:

```
┌─────────────────────────────────────┐
│      API Layer (Public)             │  ← 사용자 인터페이스
│  - KfcClient (Facade)               │
│  - suspend 함수                      │
└─────────────────────────────────────┘
           ↓
┌─────────────────────────────────────┐
│      Model Layer (Public)           │  ← 데이터 타입
│  - Data Classes                     │
│  - @Serializable                    │
└─────────────────────────────────────┘
           ↓
┌─────────────────────────────────────┐
│   Internal Implementation Layer     │  ← 내부 구현
│  - HTTP 통신                         │
│  - HTML 파싱                         │
│  - 에러 핸들링                        │
└─────────────────────────────────────┘
```

**Facade 패턴**을 사용하여 모든 API 접근은 `KfcClient` 하나로 통일됩니다.

자세한 내용은 [16-라이브러리-아키텍처.md](./16-라이브러리-아키텍처.md)를 참고하세요.

## 사용 예시

```kotlin
import dev.kairoscode.kfc.KfcClient
import kotlinx.datetime.LocalDate

suspend fun main() {
    val client = KfcClient()

    // ETF 목록 조회
    val etfList = client.getEtfList()

    // 특정 ETF 상세 정보 조회
    val etfDetail = client.getEtfDetail(
        isin = "KR7069500007",
        date = LocalDate(2024, 1, 15)
    )

    // ETF OHLCV 조회
    val ohlcvData = client.getEtfOhlcv(
        isin = "KR7069500007",
        fromDate = LocalDate(2024, 1, 1),
        toDate = LocalDate(2024, 1, 31)
    )
}
```
